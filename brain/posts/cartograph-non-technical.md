Recently added Cartograph to Alfred for Shopify — a full cart editor that opens right on the storefront.

Everything the Cart AJAX API offers, wrapped in a clean UI. Add items, change quantities, switch variants, apply discount codes, edit notes and attributes, calculate shipping rates, and view the full cart JSON for debugging.

Some things the API doesn't even support directly. Variant switching, for example — the API has no endpoint for it. Cartograph handles it behind the scenes.

I've spent too much time writing fetch calls just to do things the theme doesn't expose. This puts all of that behind buttons.

Right-click > Alfred > Cartograph, or add ?alfred=cart to any store URL.
