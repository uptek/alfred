Working on something new for Alfred for Shopify, a full cart interface that opens right on the storefront.

Everything the Cart Ajax API offers, wrapped in a clean UI.

- Add items
- Change quantities
- Switch variants: not offered by the API, I hacked together solution
- Apply discounts
- Edit notes and cart attributes
- Calculate shipping rates

I've spent far too much time writing fetch calls to interact with the cart just to do things the theme doesn't expose. This puts all of that behind buttons.

Still a work in progress.
