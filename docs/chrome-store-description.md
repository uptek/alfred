Alfred is a Shopify theme detector and productivity extension for developers and
merchants. Instantly detect themes. Streamline your workflow with smart
shortcuts and Shopify productivity tools.

### Shopify Theme Detector

Instantly identify any Shopify theme:

- Detect theme name, version, and store information
- Copy theme preview URLs with optional preview bar toggle
- Works on any Shopify storefront without external APIs

### Smart Shopify Shortcuts

Context menu shortcuts for faster navigation:

- Copy Theme Preview URL - Get shareable preview links instantly
- Open in Admin - Jump to admin pages for products, collections, pages, and
  articles
- Open in Customizer - Launch theme editor with current page loaded
- Copy Product JSON - Extract product data for development and debugging
- Copy Cart JSON - Get cart data for testing
- Clear Cart - Empty cart with one click
- Open Section in Code Editor - Navigate directly to theme section code

### Shopify Collaborator Access Presets

Save time requesting store access:

- Save frequently used permission combinations as presets
- Apply presets instantly when requesting collaborator access
- Import/export presets to share with your team
- Add custom messages to access requests

### Shopify Admin & Theme Editor Enhancements

- Collapsible admin sidebar to maximize workspace
- Resizable theme editor panels (primary sidebar, secondary sidebar, preview
  area)
- Enhanced app store partner pages with table view
- Toast notifications for better visual feedback
