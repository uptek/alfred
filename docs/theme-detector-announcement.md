Last week, I was evaluating applicants. For each Shopify store (portfolio), I'd
open Dev Tools and run Shopify.theme to identify what theme they have used.

That was a painful task for me, haha.

So I decided to build a quick Shopify theme detector into Alfred.

Now, you can navigate to any Shopify store and click the Alfred icon to see the
theme information.

You can view the store URL, theme name, and version.

It uses the Shopify.theme as source of information. So, it is prone to errors
but should work in most cases.

I hope you find it useful!
