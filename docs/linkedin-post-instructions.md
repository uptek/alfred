# LinkedIn Post Writing Instructions

## Tone & Style
- **Concise**: Keep sentences short and to the point
- **Humble**: Avoid overselling or hype language
- **Practical**: Focus on user benefits and utility over features
- **Understated**: Use softer language ("Recently added" vs "Just shipped")

## Structure
1. **Opening line**: Simple announcement of the feature
   - Be direct and clear

2. **Benefit explanation**: 1-2 sentences explaining the value
   - Focus on what the user can now do
   - Highlight problems it solves

3. **Additional context**: Optional line for extra details
   - Technical specifics (auto-open behavior, styling references)
   - User control aspects

4. **Call to action**: When including media
   - "See it in action in the video below"

## Language Preferences
- Use "Recently added" over "Just shipped"
- Be specific about user control ("Enable what you want, disable everything else")
- Reference familiar patterns when relevant ("Styled like Shopify's toast notifications")
- Avoid future promises ("what's coming next")

## Content Guidelines
- No hashtags in middle of post - remove them if initially included
- Keep paragraphs short (1-2 sentences max)
- Use line breaks for readability
- Focus on immediate, tangible benefits

## Examples of Good Phrasing
- "Gives you control over all the features"
- "No more wondering if that bug got fixed"
- "Clean, unobtrusive notifications that don't interrupt your flow"
- "To make you feel at home"

## What to Avoid
- Overly promotional language
- Long, complex sentences
- Technical jargon without context
- Future-focused promises
- Excessive enthusiasm or hype
