# LinkedIn Post - Storefront Password Auto-fill

I was getting really tired of typing the same Shopify storefront password 20+
times a day.

So I built a simple password auto-fill feature for Alfred. Now I just save the
password once, and it automatically fills and submits whenever I hit that
password page.

Here's how it works:

• Saves passwords locally in your browser (nothing leaves your machine) •
Auto-fills and submits automatically when you visit the password page • Easy to
manage – toggle it on/off, update, or delete anytime

It's such a small thing, but a big time-saver. If you're working with multiple
Shopify stores, you might find it useful too.
